# 数据类型

## 时间类型

* datetime
  * 无时区
  * 范围更广
  * yyyymm
  * 数据8个字姐
* timestamp
  * 有时区
  * 时间范围为1970-2038
  * 数据四个字节
  * 默认not null&#x20;
  * 可以指定插入和更新行为



## 字符串类型

* char
  * 固定长度
  * 更新频繁的列，char性能更好，因为不容易产生碎片
* varchar
  * VARCHAR用于存储可变长度的字符串，是最常见的字符串数据类型。它比固定长度的类型更节省空间，因为它仅使用必要的空间（即，更少的空间用于存储更短的值）。
  * 适合更新少
